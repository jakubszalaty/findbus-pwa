## Description

GraphQL Proxy for Zditm API

## Installation

```bash
yarn install
# or
npm install
```

## Start

```bash
yarn start
# or
npm run start
```

Open url: http://localhost:9000/graphiql

![](.README/2018-02-17-15-39-30.png)

## Environments

Environments are located in file `.env`.
